# Expense Tracker

A sleek and interactive web-based Expense Tracker application designed to help users manage their personal finances effectively. It provides an intuitive dashboard, visual analytics, and budget management tools to keep your spending in check.

## 🚀 Features

- **User Authentication**: Secure Sign up and Login functionality.
- **Interactive Dashboard**: A centralized, easy-to-read view of your current financial status.
- **Expense Management**: Seamlessly add, track, and categorize your daily expenses.
- **Budgeting**: Set financial goals and monitor your budgets.
- **Analytics & Visualizations**: Graphical insights into your spending habits (includes Line charts, Bar charts, and Pie charts).

## 📂 Project Structure

This project is built using standard web technologies:
- **Pages**: `home.html`, `dashboard.html`, `overview.html`, `analytics.html`, `addexpenses.html`, `budget.html`, `categories.html`
- **Authentication**: `login.html`, `login.js`, `signup.html`, `signup.js`, `auth.js`
- **Logic & Styling**: `app.js` (core logic), `style.css` (custom UI styling)

## 💻 How to Use

1. Clone or download this repository.
2. Open `home.html` or `login.html` directly in any modern web browser (Google Chrome, Firefox, Safari, Edge).
3. Create an account on the Sign Up page.
4. Log in and start tracking your expenses!
